## What we will NOT accept in this directory

- LPC (Liberated Pixel Cup) art — that is CC-BY-SA 3.0, not CC0.
- Assets labeled "free" without an explicit CC0/Unlicense/public-domain grant.
- Anything from itch.io or OpenGameArt without an explicit CC0 tag.

If you add a new tileset here, ship its license file beside it and update
this document.
